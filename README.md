# Nonesuch Formation

This repository contains data and code associated with presenting and analyzing data from the Nonesuch Formation for the publication:

Slotznick, S.P., Swanson-Hysell, N.L., Zhang, Y., Clayton, K.E., Wellman, C.H., Tosca, N.J., and Strother, P.K. (2024) Reconstructing the paleoenvironment of an oxygenated Mesoproterozoic shoreline and its record of life, GSA Bulletin, https://doi.org/10.1130/B36634.1
